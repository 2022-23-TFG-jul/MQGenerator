Bienvenido

¡Bienvenido a MQGenerator!


MQGenerator es una aplicación web diseñada para la generación automática y personalizada de problemas de control integrado de proyectos, fundamentalmente mediante técnicas de valor ganado y programación.

Una vez introducido los datos necesarios se creará un fichero .XML para su futura exportación en Moodle. Para comprobar el cuestionario puedes usar la siguiente página: https://sandbox.moodledemo.net/

Para obtener más información sobre el proceso o los autores del proyecto puedes visitar, pulsando los botones en la parte superior de esta página "Repositorio" o "Más información".

Cuando pulses el botón de Empezar se le mostrará la pantalla las diferentes tipo de preguntas CLOZE para elegir y automatizar.
